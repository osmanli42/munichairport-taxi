URL PATTERNS
================================================================================

Event Page Routes:
──────────────────────────────────────────────────────────────────────────────
{locale}/events/{slug}/

Where:
  {locale} = "de" or "en"
  {slug} = unique event identifier (lowercase, hyphens)

Examples:
  /de/events/bauma-muenchen/           ← German bauma page
  /en/events/bauma-muenchen/           ← English bauma page
  /de/events/csd-muenchen/             ← German CSD Munich
  /en/events/muenchen-marathon/        ← English Marathon
  /de/events/christkindlmarkt-muenchen/← German Christmas Market
  /en/events/tollwood-sommer/          ← English Tollwood Summer

Special Note:
  Oktoberfest uses dedicated page: /{locale}/oktoberfest/
  (NOT using the dynamic events system)
